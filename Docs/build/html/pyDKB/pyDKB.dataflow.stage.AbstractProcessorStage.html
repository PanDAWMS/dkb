pyDKB.dataflow.stage.AbstractProcessorStage module¶

Definition of an abstract class for Dataflow Data Processing Stages.

USAGE:
ProcessorStage [<options>] [<input files>]
OPTIONS:
-s, --source {f|s|h} - where to get data from: local (f)iles, (s)tdin, (h)dfs
-i, --input-dir
 DIR - base directory for relative input file names (for local and HDFS sources). If <input files> not specified, all files from the directory will be taken as the input.
-d, --dest {f|s|h} - where to send data to: local (f)iles, (s)tdout, (h)dfs
-o, --output-dir
 DIR - base directory for output files (for local and HDFS sources)
