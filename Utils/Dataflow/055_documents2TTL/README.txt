- 050_links2TTL
- 051_papers2TTL
- 052_supportDoc2TTL

from step 015_CDSDocuments, containing metadata for paper and 
corresponding supporting documents, combined from GLANCE and CDS 
metadata sources.

output of module is TTL string with metadata parameters for paper and sopporting documents
